техническая записка проекта "TaskExchange"
1. Цель

Создание веб-платформы для публикации задач, обмена решениями и их оценки сообществом.

2. Основные функции

• Регистрация/авторизация пользователей.

• Публикация задач с категориями (олимпиады, учебные курсы).

• Загрузка решений с изображениями.

• Голосование за решения (лайки/ дизлайки).

• Поиск задач по категориям.
3. Технологии

Backend: Python, Flask.

База данных: SQLite (ORM: Flask-SQLAlchemy).

Frontend: HTML/CSS, Jinja2.

• Безопасность: Хеширование паролей (Werkzeug).

4. База данных

. Таблицы:

User (пользователи).

Task (задачи).

Solution (ешения).

Vote (голоса).
